# Ammar Blueprint — Project Brief

## What this is
A personal 90-day AI automation learning platform. Single-user.
Built as a free alternative to Nick Saraev's $184/month Maker School.

## Who it's for
Ammar. Full-time SDR at SmartClick (Skopje, North Macedonia).
Currently in onboarding phase. 15-20 hrs/week available for side work.
Multilingual: Turkish (native), English (C1), Macedonian, Albanian, Serbian.

## The 90-day goal
Land first paying AI automation client ($1,500 - $3,000) targeting
Western English B2B SaaS companies.

## Strategic positioning
"B2B SaaS outbound specialist who builds AI automation systems."
Hybrid offer: strategy + execution. Not pure AAA, not pure SDR.

## Current skill stack (start of program)
- Cold outreach at user level: HeyReach, Apollo (basic-intermediate)
- Will learn through SmartClick: Clay, Instantly, Warmly
- Beginner: n8n, Make.com, Next.js, React, Git, GitHub
- Daily Claude user, learning Claude Code in Antigravity

## Curriculum philosophy
- Video-led learning (Nick Saraev, Cole Medin, Liam Ottley - all free)
- Build-by-doing, not passive consumption
- Every module ends with a shipped artifact
- 3 portfolio systems by Day 40
- Outreach starts Day 45 (not before)
- Week 0: positioning and setup before any technical learning

## Tech stack (locked for the platform itself)
- Next.js 14 (App Router)
- Tailwind CSS
- Static JSON for curriculum content (no database)
- localStorage for progress tracking
- Vercel deployment
- GitHub for version control

## Success metrics
- Day 30: 3 portfolio systems live, Upwork profile live, outreach ready
- Day 60: 10+ Upwork interviews OR 5+ cold email positive replies
- Day 90: first paying client signed ($1,500 minimum)

## Anti-patterns (what we will NOT do)
- Not buying courses (this roadmap replaces $2-5K of them)
- Not learning tools SmartClick will teach for free (Clay, Instantly, Warmly)
- Not building a 4th portfolio system before outreach (3 is enough)
- Not pitching SaaS before agency case studies (month 6+)
- Not going multi-market yet (Western English first)
- Not leaving SmartClick job (income and ICP exposure are strategic)
